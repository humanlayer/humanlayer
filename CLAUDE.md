# CLAUDE.md

Instructions for Claude when working in this repository.

## Purpose

This is a project template. Each project gets its own copy of this repo.
Claude uses this repository as context to help plan, build, and make decisions.

## Structure

- `plan.md` — project goal, stack, current status, and next steps (read this first)
- `decisions/` — record of important decisions with context and justification
- `notes/` — free-form notes, research, drafts

## How to work

- Always read `plan.md` before responding about the project
- Keep `plan.md` updated with the real current state after significant progress
- When an important decision is made, record it in `decisions/`
- Use dates in YYYY-MM-DD format

## Conventions

- Language: English
- Tone: direct and practical
- Tasks: use checkboxes `- [ ]` / `- [x]`
